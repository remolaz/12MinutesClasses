JavaScript is a client-side scripting language. The user sends an HTTP request to the server and the server send back an html file to the user that will include the javascript code. The user can read the javascript code. The user browser then executes the code when processing the page.

This is in contrast with a server-side scripting language (PHP): in this case the server won't send the code to the user but just the output of teh code. The PHP code is executed by the server.

Javascript is written inside an html file.
